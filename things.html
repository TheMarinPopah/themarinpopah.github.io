How to get any extension on your Chromebook
First, click the download button.
Second, go to chrome://extensions
Third, turn on developer mode on the top right corner of the screen
Fourth, click Load Unpacked
Fifth, find the file you downloaded earlier and click it
Finally, click open.
You should now have an extension opened up on your chromebook.
This extensions will allow you to get any extension by searching up the name followed by extension ID.
EXAMPLE: soccer Random extension ID
